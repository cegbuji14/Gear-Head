# GearHead (v1)

This project shows users which instruments are necessary to make songs in different styles according to the norms of that era.

## Tech Stack

- Node.js
- Next.js
- React
- TypeScript
- Git

## Build and Run 

Run

git clone https://github.com/cegbuji14/Gear-Head.git

cd Gear-Head

npm install

npm run dev

Open browser and go to
http://localhost:3000
